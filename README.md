# 📊 RetailPulse — Store Sales Analytics Dashboard

## 📌 Overview
RetailPulse is a Power BI dashboard designed to analyze and visualize retail store performance. It provides insights into sales, profit, customer trends, and product performance to support better business decisions.

---

## 🎯 Business Problem
Retail businesses generate large amounts of transactional data, but identifying meaningful insights can be challenging.

This project helps to:
- Track overall sales and profit performance  
- Identify top and bottom-performing products  
- Analyze customer contribution to revenue  
- Monitor sales trends over time  
- Evaluate the impact of promotions and discounts  

---

## 📂 Dataset
- Source: Microsoft Excel  
- Type: Retail transactional dataset  
- Key fields:
  - Customer ID  
  - Product Name  
  - Net Sales, Profit, Quantity  
  - City  
  - Promotion Category  

---

## 📊 Dashboard Features

### 🔹 Sales & Customer Analysis
- Net Sales by Customer ID  
- City-wise Sales Distribution  

### 🔹 Product Performance Analysis
- Top 5 & Bottom 5 Products by:
  - Sales  
  - Profit  
  - Quantity  

### 🔹 Profit Analysis
- Profit vs Net Sales relationship  
- Identification of high-sales but low-profit products  

### 🔹 Time Series Analysis
- Year-wise Sales Trend  

### 🔹 Promotion Insights
- Average Discount by Promotion Category  

---

## 📈 Key Insights
- A small number of customers contribute a major portion of total revenue  
- Top products dominate both sales and profitability  
- Some products have high sales but relatively lower profit margins  
- Sales trend shows variation across years, indicating business fluctuations  
- Promotional discounts impact sales but may reduce overall profitability

## 🚀 How to Use

1. Download the `.pbix` file from this repository
2. Open it using Microsoft Power BI Desktop
3. If prompted, update the data source path
4. Refresh the data to view the latest insights

RetailPulse enables stakeholders to gain actionable insights into sales performance, optimize product strategies, and improve overall profitability using interactive visual analytics.
